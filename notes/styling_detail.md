Design style: Dark futuristic cyberpunk UI with neon-glow accents, inspired by holographic HUD interfaces, sci-fi control panels, Tron-like lighting, and modern productivity dashboards.

Color scheme: Mostly black, charcoal, and deep navy backgrounds with high-contrast neon cyan, electric blue, hot pink/magenta, purple, and occasional lime-green accents. Use glowing outlines, soft bloom effects, and thin luminous borders. Keep the background dark and matte so the neon elements stand out.

Visual language: Use glassy panels, rounded rectangles, thin stroked borders, subtle gradients, layered cards, faint grid textures, dotted backgrounds, circuit-like details, and glowing separators. The interface should feel sleek, premium, high-tech, and slightly playful.

Typography: Mix clean futuristic sans-serif UI fonts with handwritten marker-style text for notes, sketches, brainstorming, and focus-board areas. Headings should feel sharp, digital, and bold. Handwritten sections should look like neon ink on a black glass board.

Layout style: Dense dashboard layout with multiple panels: task queue, project cards, inbox/feed, agent status, progress meters, notes, and quick actions. Everything should feel organized but energetic, like a command center for productivity and AI agents.

Mood: Focused, smart, energetic, hacker-lab, futuristic, neon workspace, cyber productivity, AI assistant dashboard. It should feel like a personal mission-control system rather than a plain task app.

Avoid: Flat boring SaaS design, white backgrounds, pastel minimalism, corporate blue-gray dashboards, overly childish gamification, or clutter that makes the UI unreadable.
